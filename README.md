 🚀 Project 1: TEXT GENERATION MODEL USING GOOGLE GEMINI (LLM)

📌 Project Title :
Cloud Comparison Generator using Gemini 3 Flash

📖 Project Description :
Developed a text-generation application using Google’s Gemini large language model to automatically generate structured and formatted comparisons between major cloud providers like AWS, Azure, and GCP.
The system uses the Gemini Flash model to generate high-quality tabular and analytical content based on user prompts.

🎯 Objective : 
To build an AI-powered text generation tool that:
Accepts user prompts
Generates structured responses (tables, summaries, comparisons)
Demonstrates prompt engineering with LLMs

🛠️ Technologies & Libraries Used :
1. Python
2. Google Generative AI SDK (google-generativeai)
3. Google Colab
4. IPython Display
5. Gemini 3 Flash Model

🧠 Model Used
 * models/gemini-3-flash-preview

⚙️ Key Responsibilities : 
* Installed and configured Gemini API
* Managed API key securely using environment variables
* Explored available models using list_models()
* Implemented text generation using generate_content()
* Designed structured prompt inputs
* Formatted output using Markdown

🖼️ Project 2: IMAGE UNDERSTANDING & TEXT EXTRACTION USING GEMINI

📌 Project Title :

AI-Powered Image-to-Text Holiday Calendar Extractor

📖 Project Description : 

Built an image understanding system using Gemini multimodal capabilities to extract structured text information from images. The model processes uploaded images and converts them into readable structured data.
The system was tested using an Indian holiday calendar image for 2026.

🛠️ Technologies & Libraries Used : 

1. Python
2. Pillow (PIL)
3. Google Generative AI SDK
4. Google Colab
5. Gemini 3 Flash Model

⚙️ Implementation Steps : 

* Loaded image using PIL
* Passed image object to generate_content()
* Generated structured holiday list output
* Converted response into Markdown

🔑 Key Responsibilities : 

1. Image preprocessing
2. Multimodal model interaction
3. Text extraction from image
4. Structured formatting of results
5. Calendar data interpretation****
